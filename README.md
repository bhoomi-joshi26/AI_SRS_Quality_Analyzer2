# AI-Based Software Requirement Specification (SRS) Quality Analyzer

## Project Title
AI-Based SRS Quality Analyzer Using NLP and Explainable AI for Early Software Defect Prevention

## Features
- SRS document upload (.txt, .docx, .pdf)
- NLP-based requirement analysis
- Ambiguity detection
- Passive voice detection
- Functional and non-functional requirement classification
- ML-based quality prediction
- Explainable AI analysis
- PDF report generation

## Technologies Used
- Python
- Streamlit
- NLP
- Scikit-learn
- Random Forest
- SHAP
- SpaCy

## Run Project

Install dependencies:

pip install -r requirements.txt

Run:

streamlit run app.py
